系统安装以及设置

1、当确定按键不在界面中时，按住Alt键就可以用鼠标拖动安装界面。

2、在25G空白磁盘上安装分区

挂载点 大小 新分区的类型 新分区的位置 文件系统
/boot 250M 逻辑分区 空间起始位置 Ext4日志文件系统
交换空间 最大1.5内存 逻辑分区 空间起始位置 交换空间
/home 10G 逻辑分区 空间起始位置 Ext4日志文件系统
/ 10G 主分区 空间起始位置 Ext4日志文件系统

3、安装完成后，先修改镜像源再修改语言，改好重启后，勾选不再询问，点击使用旧名称。

4、软件安装及卸载命令

  • 更新可用软件包列表：sudo apt update
  • 更新已安装的包：sudo apt upgrade
  • 安装软件：sudo apt install 软件名
  • 卸载软件：sudo apt remove 软件名
